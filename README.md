## Project Overview

This project uses data mining and machine learning methods to examine customer decline in the telecom sector. When consumers stop using a business, it is known as **customer churn**, which can significantly impact sales and long-term business growth.

The aim of this project is to identify customers who are at risk of switching telecom service providers by analysing historical customer data and developing predictive models. By understanding the key factors influencing churn, organisations can improve customer satisfaction and implement more effective retention strategies.

The project follows the full **business intelligence lifecycle**, including:
- Problem definition  
- Exploratory Data Analysis (EDA)  
- Predictive modelling  
- Strategic business recommendations as well as ethical, privacy, security, and technical evaluation  

The dataset used contains **7,043 customer records** and **33 attributes**, covering customer demographics, service subscriptions, billing information, and churn behaviour.
